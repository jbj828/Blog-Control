* Declaring, instanciating, initializing a 2D Array

<!-- more -->

  * Declare               ------------------------- O(1)
    * dataType[][] arr
    * ex: int[][] arr

  * Instanciation of Array -------------------------O(1)
    * arrayRefVar = new datatype[row][col];
    * ex: arr = new int[2][3];

  * Initialization        --------------------------O(mn)
    * a[0][0]=10;
    * a[0][1]=20;
    * a[0][2]=30;
    * a[1][0]=40;
      .....


  * Declaring, instanciating, initializing -----------O(1)
    * int[][] arr = {{1,2,3},{4,5,6}};
